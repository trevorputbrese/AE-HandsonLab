# Install the CF CLI Tool

Please choose the installation method below based on your Macbook's chip type.

## Option 1: For Silicon/ARM Macbooks (M1, M2, M3)
*Basically any Macbook made since late 2020.*

1. **Download the installer:**
   [Download cf10-cli-installer-darwin-arm64.pkg](https://drive.google.com/file/d/1IJLo_Nr71iDg5q5M0Yu7BFvmuf0K0V30/view?usp=drive_link)

2. **Run the installer:**
   * Navigate to your **Downloads** folder.
   * Double-click the file named `cf10-cli-installer-darwin-arm64.pkg`.
   * Accept the defaults to complete the installation.

3. **Verify installation:**
   * Open your terminal (Press `Command + Space`, type `terminal`, and press `Enter`).
   * Type the following command and hit Enter:
     ```bash
     cf
     ```
   * If the output shows a list of available `cf` commands, the installation was successful.
